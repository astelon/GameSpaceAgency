# Playtest Notes

Use this file to record outcomes of blind playtests: participants, final VP totals, key balance issues, card dominance, failure frequency, and recommended changes.

Current balance focus:
- Commercial missions should be the main source of Credits.
- Prestige missions should be the main source of VP.
- Infrastructure missions should feel like setup turns that unlock later value instead of immediate spikes.
- Persistent assets should matter: Comm Satellite and Station Hub should generate Credits, while Imaging Probe and Microgravity Lab should generate VP.

Suggested metrics to record:
- Average VP per player
- Average launches attempted per player
- Failure rate per launch (%)
- Most-used cards and win correlation
- Observed pain points (long turns, unclear text, broken combos)
- Credits earned from Commercial missions vs other sources
- VP earned from Prestige missions vs milestones and research activations
- Number of station or satellite activations per round
- Whether players understood the split between Thrust as launch capability and Range as orbital travel budget

Suggested playtest questions:
- Did Commercial missions feel necessary but not dominant?
- Did Prestige missions feel tempting enough to delay cashflow?
- Did Infrastructure missions create useful long-term value, or did they feel too slow?
- Did station operations or research assets snowball once established?
- Was the Launch Capability Check easier to understand than the previous Lift Check wording?
